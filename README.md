# SWException EML-BE


## Prerequisites
- Browser: 
  - Google Chrome
  - Firefox
- NodeJS
- Serverless
- AWS Account
- git

## Git clone

`git clone --recurse-submodules https://github.com/techsweave/eml-be.git`

Will clone all submodules

## Update submodules

`git submodule update --remote`

## Serverless deploy

`./deploy.sh`

Will deploy all services
